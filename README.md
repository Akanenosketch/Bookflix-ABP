# Bookflix-ABP

Repositorio que contiene el notebook y el dataset usados en el Proyecto Bookflix de la asignatura Aprendizaxe Baseada en Proxectos del curso 2025/2026.

Este proyecto también contiene una demo de como sería la plataforma para su defensa.

## Pasos previos

Para probar el funcionamiento del notebook, es necesario descargar el dataset data.csv (ubicado en la carpeta data/ ) y añadirlo a los archivos del notebook.
